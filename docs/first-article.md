# 第一篇文章

欢迎来到我的博客！这是第一篇文章。

## 关于本站

本站使用 [VitePress](https://vitepress.dev/) 搭建，托管在 GitHub Pages 上。

## 为什么选择 VitePress？

- ⚡ **极快**：基于 Vite 构建，开发体验流畅
- 📝 **Markdown**：纯 Markdown 写作，专注内容
- 🎨 **美观**：默认主题简洁大方
- 🔧 **可扩展**：支持 Vue 组件和自定义主题

## 开始写作

在 `docs` 目录下新建 `.md` 文件即可添加新文章。例如创建 `docs/my-post.md`，然后通过 `/my-post` 访问。

```bash
# 本地预览
npm run docs:dev

# 构建生产版本
npm run docs:build
```
